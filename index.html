Jump To Explorer

A tiny static redirector that reads #/chain/<chain>/<resource>/<id> and forwards you to the default block explorer defined by viem.

Preparing route...

Open this page with a supported hash route to redirect automatically.

Continue to explorer

Example

#/chain/1/address/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045

Supported Inputs

  • Chain IDs can be decimal like 1 or hex like 0x1.
  • Primary resources: tx, address, token, block.
  • Aliases: transaction, account, wallet, contract, erc20.
  • The target explorer uses the default explorer URL from viem/chains.
